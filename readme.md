Schema Registry
=========================
Simple Schema Registry. Stores JSON-schemas in a MongoDB. 


# Endpoints
------------
GET /schemas/<name>
GET /schemas/<name>/avro
GET /schemas/<name>/<version>
POST /schemas/<name>    
PUT /schemas/<name>
